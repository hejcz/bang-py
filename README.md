# :white_check_mark: Etap 1

Nie ma ról graczy - bandytów, szeryfa etc. Gracze mają tyle HP ile karty ich postaci (na początku 4). 
Mogą dobierać karty, strzelać, unikać i leczyć się.

| 12          | pudło       | nic                   | -     |
| 6           | piwko       | nic                   | -     |
